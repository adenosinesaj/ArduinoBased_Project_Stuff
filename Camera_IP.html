What to try if nothing loads
  1. Open these directly in your browser address bar:
    • http://192.168.0.176
    • http://192.168.0.176:81/stream
    • http://192.168.0.176/mjpeg/1
  2. Confirm your computer is on the same network as the ESP32 (same Wi-Fi).
  3. If you opened this file with file:// and it blocks images, try serving it with a simple server (instructions below).
  4. Open browser DevTools (F12) → Console and paste any errors into the log box here and send them to me.
